Curveball is a framework for building APIs in Node.js.
Curveball makes it easy to build developer-friendly HTTP APIs that are simple and robust.

Design goals

  • A small core. Curveball is a micro-framework.
  • Batteries included. Optional plugins for lots of common tasks.
  • Gets HTTP. Use modern features such as HTTP/2 Push, Content-Negotation, 1xx responses, the Prefer header and Links headers.
  • Serverless friendly. Built-in AWS Lambda support.

Installation

npm install @curveball/core

Project status

Curveball is currently in Beta. It's close to a stable release, and any API changes will be minor. It's ready for production.
